Setup Task Scheduler

> Open Task Scheduler
> Create Basic Task
> Name: Aggresand Daily DB Backup
> Trigger: Daily
> Time: e.g. 02:00 AM
> Action: Start a Program

Program/script
> C:\php\php.exe

Add arguments
> C:\Users\{user}\Desktop\aggresand-sales\scripts\daily_backup.php

Start in
> C:\Users\{user}\Desktop\aggresand-sales\scripts

CREATE TABLE delivery_payments (
    payment_id INT AUTO_INCREMENT PRIMARY KEY,
    dr_no VARCHAR(50) NOT NULL,

    si_no VARCHAR(50),
    check_no VARCHAR(50),
    date_paid DATE NOT NULL,

    amount_paid DECIMAL(12,2) NOT NULL,
    status ENUM('UNPAID','PARTIAL','PAID') NOT NULL,

    created_by INT,
    date_created DATETIME DEFAULT CURRENT_TIMESTAMP,
    is_deleted TINYINT DEFAULT 0
);
